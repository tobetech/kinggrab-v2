# ฟีเจอร์ใหม่ที่เพิ่มเข้ามา

## 1. การแสดงรายละเอียดยอดขายของแต่ละสินค้า

### วิธีใช้งาน:
- คลิกที่แถวสินค้าใดๆ ในตาราง
- จะเปิด Modal แสดงรายละเอียดยอดขายของสินค้านั้น
- Modal จะแสดง:
  - สรุปยอดรวม (จำนวนรวม, ยอดขายรวม, จำนวนรายการ)
  - ตารางรายละเอียดการขายแต่ละรายการ (วันที่, จำนวน, ยอดขาย)

## 2. การกรองยอดขายตามวันที่

### วิธีใช้งาน:
- ใช้ Date Picker ที่อยู่ด้านบนขวาของตาราง
- เลือกวันที่ที่ต้องการดูยอดขาย
- ระบบจะกรองข้อมูลให้แสดงเฉพาะยอดขายในวันที่เลือก
- คลิกปุ่ม X เพื่อล้างการกรอง

### หมายเหตุ:
- เมื่อกรองตามวันที่ Summary Cards และตารางจะแสดงเฉพาะข้อมูลในวันที่เลือก
- Modal รายละเอียดสินค้าก็จะแสดงเฉพาะข้อมูลในวันที่เลือกด้วย

## 3. การเรียงลำดับสินค้าตามยอดขาย

### วิธีใช้งาน:
- คลิกปุ่ม "สูงสุด" เพื่อเรียงสินค้าตามยอดขายจากมากไปน้อย
- คลิกปุ่ม "ต่ำสุด" เพื่อเรียงสินค้าตามยอดขายจากน้อยไปมาก
- คลิกอีกครั้งเพื่อยกเลิกการเรียงลำดับ

### หมายเหตุ:
- ปุ่มที่เลือกอยู่จะแสดงเป็นสีฟ้า (active state)
- การเรียงลำดับจะทำงานร่วมกับการกรองวันที่

## ตัวอย่างการใช้งาน

1. **ดูรายละเอียดสินค้า:**
   - คลิกที่แถว "สินค้า A" → เปิด Modal แสดงรายละเอียดการขายทั้งหมดของสินค้า A

2. **กรองตามวันที่:**
   - เลือกวันที่ 2024-01-15 → ตารางจะแสดงเฉพาะยอดขายในวันที่นั้น
   - Summary Cards จะอัปเดตตามข้อมูลที่กรอง

3. **เรียงลำดับ:**
   - คลิก "สูงสุด" → สินค้าที่มียอดขายสูงสุดจะอยู่ด้านบน
   - คลิก "ต่ำสุด" → สินค้าที่มียอดขายต่ำสุดจะอยู่ด้านบน

4. **ใช้ร่วมกัน:**
   - กรองวันที่ 2024-01-15 → เรียง "สูงสุด" → คลิกสินค้า
   - จะแสดงสินค้าที่มียอดขายสูงสุดในวันที่ 2024-01-15 พร้อมรายละเอียด
